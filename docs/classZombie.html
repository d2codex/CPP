Detailed Description

Represents a simple zombie with a name and announce behavior.

Represents a single zombie with a name.

Can be allocated on the heap with newZombie() or on the stack with randomChump(). Destructor prints a self-destruct message.

The Zombie class supports announcing itself and can be used as part of a dynamically allocated horde. In DEBUG mode, construction and destruction are tracked with static counters.
