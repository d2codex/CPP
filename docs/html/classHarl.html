Detailed Description

Simple logger that prints complaints at different levels.

Prints complaint messages at different severity levels.

The class maps a string level to a corresponding message function. Supported levels: DEBUG, INFO, WARNING, ERROR.

The class contains four message methods and a dispatcher that prints messages from a given level and above.
