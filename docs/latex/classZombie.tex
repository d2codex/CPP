\doxysubsection{Detailed Description}
Represents a simple zombie with a name and announce behavior. 

Represents a single zombie with a name.

Can be allocated on the heap with new\+Zombie() or on the stack with random\+Chump(). Destructor prints a self-\/destruct message.

The \doxylink{classZombie}{Zombie} class supports announcing itself and can be used as part of a dynamically allocated horde. In DEBUG mode, construction and destruction are tracked with static counters.
