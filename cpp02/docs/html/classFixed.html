Detailed Description

Represents a fixed-point number.

Stores the raw fixed-point value and provides accessors.

◆ Fixed() [4/4]

Fixed::Fixed ( const float  floatPoint)

Constructs a Fixed object from a float.

Converts the float to fixed-point by multiplying by 2^fractionalBits and rounding to the nearest integer.

Note
Bit shifting cannot be done directly on a float, so the float must be multiplied first, then rounded to an integer.
Parameters
floatPointValue to convert to fixed-point.
